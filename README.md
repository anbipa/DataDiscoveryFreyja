# Freyja Data Discovery Demo
This repository contains the code for demonstrating the Freyja Data Discovery tool. The demo showcases various functionalities of the Freyja Data Discovery tool, which is implemented in the following repository: [FREYJA GitHub](https://github.com/dtim-upc/FREYJA).


## Requirements
### Backend Server
Before proceeding, make sure to review the requirements and setup instructions provided in the [FREYJA repository](https://github.com/dtim-upc/FREYJA).
### Python Libraries
Ensure you have the following Python libraries installed to run the demo:

- **`requests`**: Used to send HTTP requests to the backend server.
- **`pandas`**: A powerful library for data manipulation and analysis, providing data structures like DataFrame.
- **`itables`**: A Python library that enables the creation of interactive tables for Jupyter Notebooks, enhancing data exploration.
- **`sklearn`**: A widely-used machine learning library that provides tools for data analysis, including algorithms for classification, regression, clustering, and dimensionality reduction.
## Tutorial
1. Follow the setup instructions from the FREYJA repository to set up the backend server locally using Gradle.
2. Once the server is running, execute the provided Python notebook Demo (use_case) in this repository to see the tool in action.
## Notes
- Ensure that the backend server is running locally.
- Adjust the file paths (`data_path`, `profiles_path`, etc.) as per your local setup.
